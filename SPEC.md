# WHM Alert - CI/CD Pipeline Alerting System

## Project Overview
- **Project Name**: WHM Alert
- **Type**: CLI Tool with Alerting
- **Core Functionality**: Monitors CI/CD pipelines and sends alerts when issues are detected (Slack, Discord, Email, Webhook)
- **Target Users**: DevOps teams who need automated pipeline monitoring and alerting

## Features
- Real-time pipeline monitoring
- Multi-channel alerting (Slack, Discord, Email, Webhook)
- Custom alert rules and thresholds
- Failure notifications
- Duration spike alerts
- Queue time alerts
- Configurable check intervals
